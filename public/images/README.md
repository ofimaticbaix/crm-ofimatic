# Background Image

✅ **Imagen configurada**: `background.png`

Para cambiar la imagen de fondo:

1. Coloca tu nueva imagen en esta carpeta: `public/images/`
2. Renómbrala a: `background.png` (o `background.jpg`)
3. La imagen se aplicará automáticamente al fondo del CRM

## Recomendaciones

- **Formato**: JPG, PNG o WebP
- **Resolución**: Mínimo 1920x1080px
- **Estilo**: Tech/abstracto con colores suaves para mejor legibilidad
- **Tamaño**: Optimizar para web (< 500KB recomendado)

Si no hay imagen, se mostrará un gradiente azul/morado por defecto.

Opción B · concepto + desglose

Versión desglosada (con bullets)

Concepto: Desarrollo e implantación del CRM Ofimatic Baix v1.0

Incluye:

  • Arquitectura multi-empresa Next.js 16 + Supabase PostgreSQL con seguridad a nivel de fila (RLS).
  • Autenticación, onboarding automático y panel de administración global.
  • Módulos: contactos, empresas, oportunidades (pipeline kanban con drag-and-drop), tareas y calendario interactivo.
  • Importación inteligente desde CSV, Excel y Access con auto-mapeo, deduplicación y perfiles reutilizables.
  • Geolocalización de empresas, mapa interactivo y registro de visitas comerciales con check-in.
  • Webhooks salientes vía triggers de base de datos y API REST pública v1 con claves de API.
  • Integración de pagos (Stripe), asistente IA (Anthropic Claude) y observabilidad (Sentry, PostHog).
  • Aplicación Android nativa firmada (Capacitor) con sistema de notificaciones unificado.
  • Diseño responsive con modo claro/oscuro, manual de usuario en español y despliegue en Vercel.
